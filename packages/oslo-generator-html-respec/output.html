This document describes a vocabulary, in this case Json Test. A vocabulary is a collection of reusable terms, namely classes and properties, whose meaning is fixed by means of a label and definition. Additional formal agreements are available for terms defined in this domain. For external terms, the formal conventions are taken from the external source.

4. Conformity

An exchange of data, however the exchange takes place, conforms to this vocabulary when it uses the terminology (classes and properties) in a manner consistent with the semantics established in the latest version of the specification (domain, range , definition and usage) and does not use terminology from other vocabularies as an alternative to the proposed terminology included in this vocabulary.
